# 自动化运维

Linux 命令行操作由于具备可编程，可控制的特征，天生就适合自动化管理。

Linux自动化运维技术有多种，比较流行的包括：Shell脚本、Ansible、Chef等自动化技术。

Websoft9提供的自动化部署以Ansible为核心技术，集合Shell脚本，实现复杂软件部署的全过程自动化。

同时，Linux系统也是非常重要的开发平台，延伸到开发领域的自动化，就诞生了DevOps技术：
